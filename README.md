# Authentication System Styled

A modern, responsive authentication system built with HTML, Bootstrap 5, Bootstrap Icons, and custom CSS. Features a professional color palette, Google Fonts, smooth transitions, and full responsiveness for all device sizes.

## Features
- **Login Page**: Bootstrap card, form controls, and styled buttons
- **Registration Page**: Bootstrap card, form controls, and validation
- **Forgot Password Page**: Bootstrap card, email input, and styled button
- **Reset Password Page**: Bootstrap card, password visibility toggle, and styled button
- **Dashboard Page**: Bootstrap navbar, container, and professional layout
- **Custom CSS**: Color scheme, fonts, hover effects, box shadows, background gradient, and transitions
- **Fully Responsive**: Works on desktop, laptop, tablet, and mobile

## Getting Started
1. Open `index.html` in your browser.
2. Navigate between pages using the provided links.
3. All styling is handled via Bootstrap CDN and `styles.css`.
